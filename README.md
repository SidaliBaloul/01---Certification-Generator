# 📜 Certification Generator

A simple **Certification Generator** built using **HTML, CSS, and Vanilla JavaScript**.  
This project was created as a hands-on practice to strengthen my understanding of frontend fundamentals and core programming logic.

---

## 🚀 Project Purpose

The main goal of this project is to:
- Strengthen my understanding of **JavaScript fundamentals**
- Practice **DOM manipulation**
- Improve my **frontend development skills**
- Focus on **logic building and problem-solving**
- Work without any frameworks to reinforce core concepts

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## 🎯 Features

- Generate a certificate dynamically
- Input-based customization
- Real-time DOM updates
- Simple and responsive UI
- Clean and minimal design

---

## 📚 What I Learned

While building this project, I focused on:
- JavaScript logic and functions
- Working with the DOM
- Handling user input
- Structuring a frontend project without frameworks
- Improving problem-solving skills through real use cases

---

## 💡 Why Vanilla JS?

This project was intentionally built without any frameworks to:
- Understand how JavaScript works under the hood
- Build strong programming foundations
- Avoid abstraction and focus on core logic
